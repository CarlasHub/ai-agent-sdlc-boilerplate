# Data Classification

Highest data class allowed: public

Data sources:

Local project files, local documentation templates, fictional sample data, package scripts, test notes and approved project documentation.

Personal data involved: no

Secrets involved: no

Data restrictions:

Real candidate data, real client data, real employee data, personal data, secrets, API keys, credentials, production data and confidential company information must be blocked.

Data owner:

Project owner and AI governance reviewer.
