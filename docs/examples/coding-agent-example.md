# Example: Coding Agent

## Approved purpose

Assist with scoped code changes from approved tickets.

## Allowed

- read approved source files
- suggest implementation
- write approved files after gate
- generate tests
- create draft PR if approved

## Not allowed

- push to main
- deploy
- access secrets
- delete files
- change unrelated architecture
- approve its own PR

## Required evidence

- files changed
- tests run
- risks
- human reviewer
