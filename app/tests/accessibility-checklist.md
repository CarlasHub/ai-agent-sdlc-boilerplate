# Accessibility Checklist

- [ ] Page has one clear `h1`.
- [ ] Interactive controls have accessible names.
- [ ] Project type selection is keyboard reachable.
- [ ] Form labels are programmatically associated with inputs.
- [ ] Colour is not the only way to communicate status.
- [ ] Focus states are visible.
- [ ] Layout works at mobile and desktop widths.
- [ ] Reduced-motion mode avoids non-essential animation.
